# ebike-store-bidasar
A modern, responsive, and SEO-optimized dealership website for E-Bike Store Bidasar (Authorized Tunwal E-Motors Dealer). Built with React, Vite, Tailwind CSS, and Framer Motion. Features scooter catalog, finance/EMI information, gallery, contact form, WhatsApp integration, responsive design, and production-ready deployment on Vercel.
